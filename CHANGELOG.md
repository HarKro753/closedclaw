# Changelog

## [1.0.0] - 2026-02-24

### Added
- User authentication (signup/login with email + password)
- Personal AI agent provisioned at signup with isolated workspace and memory
- Chat interface with conversation history
- Agent tools: read/write memory, read/write/list files, web search
- Admin panel for user and agent management
- SQLite storage with better-sqlite3
- Docker Compose setup for self-hosting
- JWT-based authentication with 7-day expiry
- Admin role assignment via ADMIN_EMAILS environment variable
- Responsive Next.js 15 dashboard with Tailwind CSS
